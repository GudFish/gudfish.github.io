# 主机网络代理与虚拟机共享
(1) 首先在代理软件的设置中打开局域网共享，查看端口，如7897，并查看主机与虚拟机所在的局域网段，`ifconfig`，如`192.168.14.1`；

(2) 在虚拟机(如ubuntu)网络设置中，选择"网络代理"->"手动"，为需要的代理添加`http://192.168.14.1`，端口设为上述端口7897，设置HTTP和HTTPS即可浏览器访问了，忽略主机写入`localhost, 127.0.0.0/8, ::1`；

(3) 如果需要代理github，添加
```shell
git config --global http.proxy http://192.168.14.1:7897
git config --global https.proxy http://192.168.14.1:7897
```
